# 🚗 Car Showroom Management System

## Project Overview

This project is a Java Object-Oriented Programming (OOP) application that simulates a car showroom management system.

The system manages both petrol and electric cars, calculates their prices, identifies the most expensive car, and computes the total inventory value of the showroom using core OOP concepts.

## Features

- Manage Petrol and Electric Cars
- Display Complete Car Information
- Calculate Car Prices
- Differentiate Luxury and Standard Petrol Cars
- Differentiate Long Range and City Electric Vehicles
- Find the Most Expensive Car
- Calculate Total Showroom Inventory Value

## Technologies Used

- Java
- Object-Oriented Programming (OOP)

## OOP Concepts Used

- Abstraction
- Inheritance
- Encapsulation
- Polymorphism
- Method Overriding
- Arrays of Objects

## Sample Output

The program displays:

- Car Details
- Engine CC / Battery Capacity
- Car Type
- Final Price
- Most Expensive Car
- Total Showroom Inventory Value
